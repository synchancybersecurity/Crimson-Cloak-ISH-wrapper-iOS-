# Crimson-Cloak-ISH-wrapper-iOS

Wrapper to maximize iSH iOS app capabilities without jailbreak.

## What It Does
- HTTP API + embedded dashboard on `http://127.0.0.1:8088`
- WebSocket real-time streaming on `ws://127.0.0.1:8089`
- Auto-discovery of SynChan tools (Huginn, OmnINT, Daedalus, Argus, Neural, Ophelia, Hecate, MIDAS, SynChan, Crimson Cloud)
- Per-tool shortcut buttons
- Clipboard sync, file watcher, network monitor, keepalive
- SSH reverse tunnel auto-connect to Kali/Pi
- iOS Shortcuts integration
